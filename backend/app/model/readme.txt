Перед использование системы однократно выполните файл "init.sh". Ваш исполняемый файл должен находится в одной директории с сгенерированной скриптом папкой "ru-gpts". На рабочей машине должна быть установлена CUDA. Как использовать модуль смотрите в примере или используйте docstring. Приятного пользования :)
